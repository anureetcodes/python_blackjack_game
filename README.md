Python Blackjack Project

A command line implementation of the classic Blackjack (21) card game built with Python. This project was developed as part of my 100 days of Python bootcamp learning journey to strengthen my understanding of programming fundamentals and problem solving.

 Project Overview
In this game, the player competes against the computer (dealer) by drawing cards and trying to reach a score as close to 21 as possible without exceeding it. The game follows the basic rules of Blackjack, including special handling of Aces and dealer decision-making.

 Features

- Random card generation
- Automatic score calculation
- Ace value adjustment (1 or 11)
- Dealer follows Blackjack rules
- Detects Blackjack, win, lose, and draw conditions
- Option to play multiple rounds

 Technologies Used

- Python 3
- Random Module

Concepts Practiced

- Functions
- Conditional Statements
- Loops
- Lists
- Variables
- User Input
- Game Logic
- Problem Solving & Debugging

   How to Run

1. Clone this repository.
2. Open the project in your preferred Python IDE (PyCharm, VS Code, etc.).
3. Run the main Python file.

 What I Learned

Building this project helped me:

- Break a larger problem into smaller functions.
- Write cleaner and more organized Python code.
- Improve debugging and logical thinking.
- Understand how multiple programming concepts work together in a complete project.

 Future Improvements
